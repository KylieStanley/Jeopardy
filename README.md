# Jeopardy
 
## Set Up

Clone down this repo

```shell
  git clone git@github.com:KylieStanley/Jeopardy.git
```

Change into Jeopardy directory
  
```shell
  npm install
```

## Testing
Mocha and Chai is used for testing

In the Jeopardy directory:

```shell
npm test
```
